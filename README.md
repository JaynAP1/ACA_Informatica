# ACA_Informatica

Este proyecto es una página web informativa sobre el tema "La informática como centro cultural". Su propósito es presentar el contenido académico del proyecto ACA mediante una estructura clara, visual y fácil de navegar.

## ¿Qué incluye?

- Una página principal con información general del proyecto.
- Secciones sobre los módulos de estudio:
  - Creación y gestión digital
  - Colaboración web y socialización virtual
  - Seguridad y tecnologías emergentes
- Recursos integrados como documentos PDF y una presentación en formato PPTX.
- Diseño sencillo y responsivo para visualizarse en distintos dispositivos.
